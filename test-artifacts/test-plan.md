# Test Plan: Automation Exercise Playwright SQA Framework

## Objective
Validate key user journeys and API behavior of the Automation Exercise e-commerce practice application using Playwright and TypeScript.

## Scope
- Home page validation
- User registration
- Login/logout
- Product listing
- Product search
- Product details
- Category and brand filtering
- Cart add/remove
- Checkout review
- Contact form submission
- Products, brands, search, and authentication API tests

## Out of Scope
- Real payment processing
- Performance testing
- Security testing
- Accessibility testing beyond basic visible checks

## Test Types
- Smoke testing
- Regression testing
- Cross-browser UI testing
- API testing
- Defect evidence testing

## Browsers
- Chromium
- Firefox

## Tools
- Playwright
- TypeScript
- Node.js
- GitHub Actions
- HTML Report
- Screenshots/videos/traces on failure
