# Bug Report Samples

## BUG-001: Product search returns unrelated results

**Environment:** Chrome, Windows 11  
**Severity:** Medium  
**Priority:** High  
**Steps to Reproduce:**
1. Open Products page
2. Search for "Blue Top"
3. Check search results

**Expected Result:** Only relevant products are displayed.  
**Actual Result:** Unrelated products appear.  
**Evidence:** Screenshot, video, trace from Playwright report.

## BUG-002: Cart count does not update after adding product

**Environment:** Firefox, Windows 11  
**Severity:** High  
**Priority:** High  
**Steps to Reproduce:**
1. Open Products page
2. Add one product to cart
3. Open cart page

**Expected Result:** Cart shows exactly one product.  
**Actual Result:** Cart count is incorrect.  
**Evidence:** Screenshot, video, trace from Playwright report.
